타임리프 예제 2

요청 위임된 request에 존재하는 parameter 출력하기

${param.key}

- request에 존재하는 parameter값 얻어와 출력
- 요청 URL에 Query String(GET)이나 폼 데이터(POST)로 전달된 parameter값을 직접 req에 세팅하지 않아도 바로 접근할 수 있도록 해 줌.
  • 제출된 이름
  • 제출된 나이
  • 체크된 색상

th:text / th:utext

th:text = "속성값"
- 해당 태그에 "속성값"을 내용으로 출력
- 단, html 태그, 특수 문자 해석 불가 (innerText)

th:utext = "속성값"
- 해당 태그에 "속성값"을 내용으로 출력
- 단, html 태그, 특수 문자 해석 가능 (innerHTML)

th:text (HTML 태그 해석 x)

str

th:utext (HTML 태그 해석 o)

str

th:text / th:utext 출력 시 변수에 저장된 값이 아닌 임의의 내용 작성해보기 

th:text="|문자열 ${}|"
- 해당 요소의 내용으로 "문자열" + 변수값 출력 가능

또는
+ 연산으로도 이어쓰기 가능
