${#numbers.sequence(시작, 끝 [,step])} : 번호 발생기

- 시작부터 끝까지 번호로만 이루어진 숫자 배열 생성
  ex) ${#numbers.sequence(1,5)} => [1,2,3,4,5] (step 기본값 == 1)
  
<사용 이유>
- Thymeleaf 반복문 th:each는 향상된 for문 형태로 일반 for문의 동작을 못하기 때문에 이를 해결하기 위해 사용
